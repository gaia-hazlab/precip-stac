# precip-stac
precipitation product stac catalog

Example catalog for UW Atmospheric River Precipitation Analysis

From https://orca.atmos.washington.edu

This is a quick example for making some of the precipitation products from the UW Atmospheric River Precipitation Analysis available as a STAC catalog. See https://orca.atmos.washington.edu/~bkerns/realtime/ar/ for static plot animations of multiple layers.
